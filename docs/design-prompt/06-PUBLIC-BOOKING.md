# PraktiQU - Public Booking Page (Client-Facing)
## Design Prompt

---

## Design Philosophy

- Clean, minimal, trust-building design
- Mobile-first (clients likely book on phone)
- Calming color palette (not too clinical)
- Simple, non-technical language
- Privacy-focused messaging

---

## Color Palette (Client-Facing)

| Color | Hex | Usage |
|-------|-----|-------|
| Primary | #5046E5 | CTAs, progress |
| Primary Light | #EEF2FF | Backgrounds |
| Success | #22C55E | Confirmations |
| Background | #FAFBFF | Page background |
| Surface | #FFFFFF | Cards |
| Text Primary | #1E293B | Main text |
| Text Secondary | #64748B | Secondary |

## Responsive Behavior

| Screen | Layout |
|--------|--------|
| Mobile (<640px) | Full-width cards, stacked layout |
| Tablet (640-1024px) | Side-by-side professional cards |
| Desktop (>1024px) | Centered container, max-width 480px for form |

## Generate these screens:

1. **Welcome/Landing** - Public booking homepage
2. **Select Professional** - Professional selection
3. **Select Service** - Service type selection
4. **Select Date Time** - Calendar and time picker
5. **Client Information Form** - Contact details
6. **Success Confirmation** - Booking confirmation
7. **Login Page** - For existing clients
8. **Responsive Mobile View** - Mobile-optimized layouts
