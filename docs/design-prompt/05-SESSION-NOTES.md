## Session Note Tabs Detail

### Subjective Tab
- Client's chief complaint
- Main concerns
- Progress since last session
- Client's goals
- What worked/not worked

### Objective Tab
- Behavioral observations
- Mental status (if applicable)
- Vital signs (optional: mood scale)
- Non-verbal cues
- Session dynamics

### Assessment Tab
- Clinical interpretation
- Diagnosis considerations
- Progress indicators
- Risk assessment
- Prognosis

### Plan Tab
- Homework assignments
- Interventions to try
- Follow-up schedule
- Referrals if needed

## Generate these screens:

1. **Session Notes Editor** - Main SOAP editor
2. **Session Notes View** - Read-only completed notes
3. **Intervention Plan List** - All plans for client
4. **Add Intervention Plan Modal** - Create new plan
5. **Intervention Plan Detail** - Single plan view with progress
6. **Session Notes History** - List of all notes for client
