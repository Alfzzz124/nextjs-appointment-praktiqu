# Work Log

## Purpose

This directory stores milestone-level work log entries. For routine changelog-style entries, use git commit history instead.

## Format

### YYYY-MM-DD - Milestone Title

**Summary**
Brief description of work completed.

**Key Changes**
- List of significant changes or deliverables

**Artifacts**
- Links to specs, PRs, or documentation

**Lessons Learned**
- Any durable insights from this work
