# Feature: Session Notes Templates

**Input**: Pre-defined templates for session notes structure

## Core Artifacts

- spec.md: Template management, SOAP/DAP structure, template selection
- plan.md: Template model, template-per-service association
- tasks.md: Template CRUD, template selection at session start
- memory-synthesis.md, checklists/requirements.md

## Key Concept

Professionals select a template when starting notes. Templates define structure (headings/sections). No template content in session notes themselves.

## Requirements

- US-07.06: Use notes template
- Templates linked to service types (KONSELING/ASESMEN/WORKSHOP)
- Constitution: TDD, Design-Driven
